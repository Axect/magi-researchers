# Journal Strategy Domain Context — Particle Physics Phenomenology

You are assisting with journal selection and publication strategy for a particle physics phenomenology research project. Apply the following principles throughout all phases:

## Core Principles
- **Framing Is Everything**: The same physics result requires fundamentally different presentation for different journals. PRL demands broad-impact narrative; PRD demands technical depth; JHEP demands method clarity and community utility.
- **Pre-Commit the Cascade**: Define your backup submission path before submitting to your first-choice journal. This reduces post-rejection delays and emotional decision-making.
- **Referee Defense First**: Most rejections are preventable. Anticipate reviewer objections and address them explicitly before submission.
- **Career-Aware Strategy**: Early-career researchers should optimize for acceptance probability and speed; established researchers can take higher-risk, higher-reward shots.
- **ArXiv Is the Real Publication**: In HEP, the arXiv posting is the community publication event. The journal is for the formal record and CV. Plan both strategically.

## 1. Strategy Configuration

Before selecting a journal, establish the research profile:

### Paper Type Classification
- **Discovery / New Physics**: Qualitatively new phenomenon, model, or mechanism → PRL, PRX
- **Precision / Improvement**: Quantitative refinement of known results (higher-order corrections, improved fits) → PRD, JHEP
- **Method / Tool**: New computational technique, software, or framework → JHEP, PRD, Computer Physics Communications
- **Reinterpretation / Recast**: New constraints from existing experimental data → PLB, EPJC, PRD
- **Review / Comprehensive Study**: Broad survey or combined analysis → PRD, EPJC

### Sub-Field Routing
| Sub-Field | Primary Journals | Secondary Journals | Notes |
|:---|:---|:---|:---|
| BSM Model Building | PRD, PLB | PRL (if breakthrough), JHEP | PRL only if genuinely new paradigm |
| Dark Matter (theory) | PRD, JHEP | PRL (strong detection prospect), PLB | Emphasize detection signatures for PRL |
| Collider Phenomenology | JHEP, PRD | PRL (discovery-level), PLB | JHEP if method-heavy; PLB if timely |
| Neutrino Physics | PRD, JHEP | PRL (oscillation breakthroughs) | Strong experimental connection helps |
| Flavor Physics | PRD, JHEP | PLB, PRL | B-physics anomalies can push to PRL |
| QCD / Perturbative | JHEP, PRD | PRL (new technique) | JHEP preferred for multi-loop calculations |
| EFT / SMEFT | JHEP, PRD | PRL (if model-independent breakthrough) | Present model-independent results first |
| Cosmological Implications | PRD, JCAP | PRL (if cross-disciplinary impact) | Consider JCAP for pure cosmo |
| Astroparticle Physics | JCAP, PRD | PRL, PLB | JCAP is the natural home |
| Formal / Amplitudes | JHEP, NPB | PRL (elegant result) | NPB for mathematical depth |
| Computational / Numerical | CPC, PRD | PRE, JHEP | CPC if releasing code/tool |
| Statistical Mechanics (HEP) | PRE, PRD | JHEP, NPB | PRE for lattice stat-mech crossover |
| Quantum Information in HEP | PRX Quantum, PRL | PRD, PRX | PRX Quantum for quantum-native work |
| Machine Learning in HEP | PRD, JHEP | MLST, CPC | CPC if tool-oriented; MLST if ML-method-focused |

### Career Mode
- **"High-Upside" Mode** (Tenured / Established): Target PRL/PRX first. Accept longer review cycles. Optimize for citation impact.
- **"Risk-Balanced" Mode** (Postdoc / Late PhD): Target PRD/JHEP primarily. Use PRL only for clear breakthrough results. Optimize for speed and certainty — publications before Fall job cycle deadlines.
- **"Speed" Mode** (Time-sensitive / Anomaly response): Target PLB first. Fastest review turnaround. Follow up with comprehensive PRD paper.

## 2. Journal Profiles Database

### PRL (Physical Review Letters)
- **Publisher**: APS | **IF**: ~8.6 | **OA**: SCOAP3 | **Acceptance**: ~10-20%
- **Length**: 3750 words (strict), ~4 pages. >12% over → desk rejection without review
- **Scope**: All physics. Must demonstrate broad impact beyond specialist audience.
- **Acceptance Criteria** (must meet ≥1):
  1. Opens a new research area or new avenue within established area
  2. Solves or makes essential steps toward solving a critical problem
  3. Introduces techniques/methods with highly significant impact
  4. Is of unusual intrinsic interest to PRL's broad readership
- **What Editors Look For**: Clear "why should a non-specialist care?" narrative. Strong first paragraph. Self-contained figures.
- **Common Rejection Reasons**: "Incremental advance", "too specialized", "would be better suited for PRD", exceeds page limit
- **Cover Letter Must**: Explicitly state which acceptance criterion the paper meets and why

### PRD (Physical Review D)
- **Publisher**: APS | **IF**: 5.3 | **OA**: SCOAP3 | **Acceptance**: ~60-70%
- **Length**: No strict limit (typically 10-30 pages)
- **Scope**: Particle physics, field theory, gravitation, cosmology
- **Sub-sections**: Particles & Fields, Gravitation & Cosmology
- **Strengths**: Comprehensive technical work, detailed derivations, full phenomenological studies
- **What Editors Look For**: Technical rigor, complete analysis, proper benchmarking against existing results
- **Note**: Has "Rapid Communications" format for timely shorter papers (~4 pages)
- **Common Rejection Reasons**: Insufficient novelty, incomplete analysis, failure to compare with existing literature

### PRX (Physical Review X)
- **Publisher**: APS | **IF**: 15.7 | **OA**: Full open access | **Acceptance**: Very low (<10%)
- **Length**: Flexible
- **Scope**: All physics with emphasis on interdisciplinary innovation
- **What Editors Look For**: Paradigm-shifting work that bridges physics sub-fields. "Would a condensed matter physicist learn something from this particle physics paper?"
- **Best For**: Cross-disciplinary methods (e.g., quantum information meets HEP, ML-driven discoveries)
- **Common Rejection Reasons**: "Excellent but not sufficiently cross-disciplinary", "PRL or PRD would be more appropriate"

### JHEP (Journal of High Energy Physics)
- **Publisher**: SISSA/Springer | **IF**: ~5.4 | **OA**: SCOAP3 | **Acceptance**: ~50-60%
- **Length**: No strict limit
- **Scope**: Elementary particle physics, QFT, string theory, collider physics, astroparticle physics
- **Strengths**: Fast review (~2-3 months), strong in technical/methodological work
- **What Editors Look For**: Technical precision, community utility, well-defined methodology
- **Best For**: Multi-loop calculations, EFT analyses, phenomenological tools, lattice results
- **Common Rejection Reasons**: "Too phenomenological without technical depth", incomplete error analysis

### PLB (Physics Letters B)
- **Publisher**: Elsevier | **IF**: 4.3 | **OA**: SCOAP3 | **Acceptance**: ~40-50%
- **Length**: Short (typically ≤6 pages, ~3000-4000 words)
- **Scope**: Rapid publication of important results in particle, nuclear, and high-energy physics
- **Strengths**: Fast review turnaround, good for timely results
- **What Editors Look For**: Timeliness, clear result, concise presentation
- **Best For**: Quick phenomenological studies, responses to new experimental data, preliminary BSM analyses
- **Common Rejection Reasons**: "Too brief to be convincing", "needs more detail — consider PRD"

### EPJC (European Physical Journal C)
- **Publisher**: Springer | **IF**: 4.8 | **OA**: SCOAP3 | **Acceptance**: ~50-60%
- **Length**: Flexible
- **Scope**: Experimental and theoretical particle physics, including detector and accelerator physics
- **Strengths**: Broad scope, good for combined experimental+theoretical studies
- **What Editors Look For**: Solid methodology, clear presentation, relevance to ongoing experiments
- **Best For**: Comprehensive phenomenological surveys, experimental reinterpretations, detector-related theory
- **Note**: Internal transfer to EPJ+ journals possible

### NPB (Nuclear Physics B)
- **Publisher**: Elsevier | **IF**: ~2.8 | **OA**: SCOAP3
- **Length**: Flexible
- **Scope**: HEP phenomenology, theory, experiment, mathematical physics, statistical physics
- **Strengths**: Welcomes mathematically rigorous formal theory work
- **Best For**: Formal QFT, amplitudes, mathematical methods in physics, solvable models

### JCAP (Journal of Cosmology and Astroparticle Physics)
- **Publisher**: IOP/SISSA | **IF**: ~5.9 | **OA**: SCOAP3
- **Length**: No strict limit
- **Scope**: Theoretical, observational, experimental, and computational cosmology and astroparticle physics. Covers dark matter/energy theory and detection, CMB, gravitational waves, baryogenesis, inflation, neutrino cosmology, and high-energy astrophysics.
- **Strengths**: The dedicated venue for cosmology and astroparticle physics. Strong overlap with HEP phenomenology when cosmological implications are central.
- **What Editors Look For**: Clear observational/experimental predictions or confrontation with data (Planck, LHC, direct detection experiments)
- **Best For**: Dark matter relic density calculations, baryogenesis models, gravitational wave signatures from phase transitions, CMB constraints on BSM physics, astroparticle detection prospects
- **Common Rejection Reasons**: "Purely theoretical with no observational consequence", overlap with existing PRD/JHEP work without cosmological focus

### PRE (Physical Review E)
- **Publisher**: APS | **IF**: ~2.4 | **OA**: Hybrid
- **Length**: No strict limit
- **Scope**: Statistical, nonlinear, biological, and soft matter physics. Includes computational physics, complex systems, fluid dynamics, plasma physics, networks, and chaos.
- **Strengths**: Broad scope covering computational and statistical physics. Good venue for lattice/Monte Carlo methods, phase transitions, and complex systems with physics connections.
- **What Editors Look For**: Physical insight from computational or statistical methods. Clear connection to physical phenomena.
- **Best For**: Monte Carlo methods applied to physical systems, network/complex systems in physics, statistical mechanics of gauge theories, computational fluid dynamics, nonlinear dynamics in physical systems
- **Common Rejection Reasons**: "Better suited for PRD" (if too HEP-specific), "insufficient physical insight" (if purely computational)
- **Note**: Consider PRE when your work bridges particle physics with statistical mechanics or computational methods

### CPC (Computer Physics Communications)
- **Publisher**: Elsevier | **IF**: ~8.2 | **OA**: Hybrid
- **Length**: Flexible (research papers) or program descriptions (with mandatory code submission)
- **Scope**: Computational methods and techniques in physics and physical chemistry. Publishes both research papers on algorithms/methods and computer program descriptions with source code.
- **Strengths**: The premier venue for physics software and computational tools. Very high IF for a methods journal. Requires code submission for program papers.
- **What Editors Look For**: Novel computational methods with clear physics applications. For program papers: well-documented, publicly available code with installation instructions, test cases, and benchmarks.
- **Best For**: Monte Carlo event generators (MadGraph, Sherpa extensions), numerical solvers for physics problems, HEP analysis frameworks, symbolic computation tools, Feynman diagram calculators, parameter scanning tools
- **Mandatory for Program Papers**:
  - Source code submission
  - README with installation and execution instructions
  - Sample input/output for test runs
  - User manual (where appropriate)
- **Common Rejection Reasons**: "Code not submitted or not runnable", "insufficient documentation", "method lacks novelty over existing tools"

### PRX Quantum
- **Publisher**: APS | **IF**: ~11.0 | **OA**: Full open access
- **Length**: Flexible
- **Scope**: Core quantum information science and technology — quantum computing, quantum communication, quantum sensing, quantum simulation, and quantum error correction. Emphasizes milestone achievements with lasting impact.
- **Strengths**: Very high impact for quantum-specific work. Selective like PRX but focused on quantum. Open access.
- **What Editors Look For**: Significant advances in quantum science. Papers that will be cited across quantum computing, quantum physics, and quantum engineering communities.
- **Best For**: Quantum algorithms for HEP simulations, quantum error correction advances, quantum sensing for fundamental physics, quantum simulation of lattice gauge theories, entanglement in quantum field theory
- **Common Rejection Reasons**: "Incremental improvement", "too application-specific without broad quantum insight", "better suited for PRA or PRResearch"
- **Note**: High APC (~$2,700). Consider if the work genuinely advances quantum science, not just applies quantum methods.

## 3. Framing Switchboard

### Same Result, Different Framing

**Example**: You calculated NLO corrections to a BSM process at the LHC.

#### PRL Framing (Broad Impact + Urgency)
> "We demonstrate that next-to-leading-order corrections dramatically alter the discovery reach for [model] at the LHC, shifting the exclusion boundary by [X]%. This result has immediate implications for the interpretation of ongoing ATLAS/CMS searches and fundamentally changes the phenomenological landscape of [BSM scenario]."
- **Emphasis**: Discovery implications, experimental relevance, broad physics community interest
- **Structure**: Result first → Physics impact → Brief method → Conclusion with future outlook
- **Key phrases**: "broad implications", "discovery potential", "fundamentally alters", "immediate experimental relevance"

#### PRD Framing (Technical Completeness)
> "We present a comprehensive calculation of NLO QCD+EW corrections to [process] within [model]. We provide detailed numerical results including scale and PDF uncertainties, compare with existing LO results, and derive updated exclusion limits using the latest LHC Run 3 data."
- **Emphasis**: Technical rigor, completeness, reproducibility, comparison with existing work
- **Structure**: Introduction → Model setup → Calculation details → Numerical results → Phenomenological analysis → Conclusion
- **Key phrases**: "comprehensive analysis", "detailed calculation", "systematic uncertainties", "updated constraints"

#### JHEP Framing (Method + Community Utility)
> "We develop an automated framework for NLO calculations in [model class] and apply it to [specific process]. Our approach extends existing tools and provides publicly available code for the community. We present benchmark results and validate against known limiting cases."
- **Emphasis**: Methodology, tools, community utility, technical precision
- **Structure**: Introduction → Framework/Method → Implementation → Validation → Application → Conclusion
- **Key phrases**: "systematic framework", "publicly available", "benchmark results", "community tool"

### Dual-Submission Strategy (Letter + Companion Paper)
When your result warrants it, prepare both simultaneously:
- **PRL Letter**: Highlight the punchline. 3750 words. Focus on physics impact.
- **PRD Companion**: Full technical details, derivations, supplementary plots. Reference the PRL.
- **Timing**: Submit both on the same day. State in PRL cover letter: "A companion paper with full technical details has been submitted to PRD [arXiv:XXXX.XXXXX]."
- **LaTeX Strategy**: Write the PRD paper first. Extract the core narrative into the PRL draft. Move derivations to Supplemental Material or reference the companion.

## 4. Pre-Submission Checklist: Referee Defense

### Critical Checks (Must Address Before Any Submission)
- [ ] **Gauge invariance**: Is the model/Lagrangian gauge-invariant? Are all vertices consistent?
- [ ] **Unitarity bounds**: Are perturbative unitarity constraints respected in the parameter space shown?
- [ ] **Perturbativity**: Are all couplings within perturbative regime? Landau pole checked?
- [ ] **Electroweak precision**: Are S, T, U parameters (or oblique corrections) consistent with data?
- [ ] **Vacuum stability**: Is the electroweak vacuum stable (or sufficiently long-lived) in shown parameter space?
- [ ] **Flavor constraints**: FCNC bounds, meson mixing, rare decays — all compatible?
- [ ] **Direct search limits**: Latest LHC direct search bounds properly recasted and applied?
- [ ] **Cosmological constraints**: Relic density, BBN, CMB — consistent if the model has cosmological implications?
- [ ] **Existing literature**: Are you aware of and citing closely related prior work? Have you checked INSPIRE/arXiv?

### Important Checks (Strengthen the Paper)
- [ ] **Limiting cases**: Does your result reduce to known results in appropriate limits?
- [ ] **Scale dependence**: Renormalization/factorization scale uncertainties quantified?
- [ ] **PDF uncertainties**: If collider process, PDF uncertainties included?
- [ ] **Statistical treatment**: Are exclusion limits properly derived (CLs method, etc.)?
- [ ] **Benchmark points**: Are representative benchmark points provided for experimentalists?
- [ ] **Code availability**: Is simulation/analysis code available or documented?

### Journal-Specific Compliance
#### PRL
- [ ] Word count ≤ 3750 (including abstract, references count separately)
- [ ] ≤ 4 figures (combined; each panel counts)
- [ ] Cover letter explicitly states which acceptance criterion is met
- [ ] Abstract ≤ 600 characters
- [ ] No appendices (use Supplemental Material instead)

#### PRD
- [ ] Follows PRD section conventions
- [ ] Includes comprehensive bibliography
- [ ] Numerical results reproducible from description

#### JHEP
- [ ] LaTeX uses JHEP style file
- [ ] References in JHEP format
- [ ] Acknowledgments include proper funding agency format

#### PLB
- [ ] Paper length ≤ 6 pages
- [ ] Clear, concise result statement in abstract
- [ ] Timely relevance established in introduction

## 6. Cascade Submission Roadmap

### Primary Cascade Paths

```
HIGH IMPACT PATH:
PRL ──rejected──┬── "lack of broad interest" ──→ PLB (fast, keep short)
                ├── "needs more detail"       ──→ PRD (expand to full paper)
                └── "too specialized"         ──→ JHEP or PRD

INTERDISCIPLINARY PATH:
PRX ──rejected──┬── "not cross-disciplinary enough" ──→ PRL
                └── "excellent but focused"          ──→ PRD or JHEP

STANDARD PATH:
JHEP ──rejected──┬── "scope mismatch"    ──→ PRD or EPJC
                 └── "insufficient depth" ──→ PRD (expand analysis)

RAPID PATH:
PLB ──rejected──┬── "needs more detail"  ──→ PRD
                └── "insufficient novelty"──→ EPJC or NPB

COSMO/ASTROPARTICLE PATH:
JCAP ──rejected──┬── "too HEP-focused"     ──→ PRD or JHEP
                 └── "insufficient data"    ──→ PRD (add more phenomenology)

COMPUTATIONAL / TOOL PATH:
CPC ──rejected──┬── "method not novel"     ──→ PRD or JHEP (reframe as physics paper)
                └── "code insufficient"    ──→ Fix code, resubmit CPC or try PRE

QUANTUM PATH:
PRX Quantum ──rejected──┬── "not quantum enough"  ──→ PRL or PRD
                        └── "incremental"          ──→ PRA or Phys. Rev. Research
```

### Transfer Mechanisms
- **APS Internal**: PRL → PRD transfer preserves referee reports (request in cover letter)
- **Springer**: EPJC → other EPJ journals possible through editorial office
- **Cross-publisher**: No automatic transfer. Resubmit fresh but mention prior review in cover letter.

### Rewrite Deltas Per Cascade Step
| Transition | Required Changes |
|:---|:---|
| PRL → PRD | Expand all sections, add full derivations, more plots, detailed appendix |
| PRL → PLB | Minor: adjust framing from "broad impact" to "timely result" |
| PRX → PRL | Remove interdisciplinary framing, sharpen specialist narrative |
| JHEP → EPJC | Minimal: adjust style file, reframe for broader particle physics audience |
| JHEP → PRD | Adjust style file, minor framing changes |
| JCAP → PRD | Reduce cosmological focus, expand particle physics analysis |
| CPC → PRD/JHEP | Remove code-submission focus, reframe as physics result |
| PRX Quantum → PRL | Condense, broaden narrative beyond quantum community |
| PRX Quantum → PRA | Lower bar, keep quantum focus, add more technical detail |

## 7. ArXiv Strategy Guide

### Category Selection
| Primary Category | When to Use |
|:---|:---|
| `hep-ph` | Default for phenomenology: BSM, collider pheno, dark matter theory, flavor, neutrino theory |
| `hep-ex` | Only if the paper has strong experimental collaboration involvement |
| `hep-th` | Formal theory aspects: amplitudes, EFT foundations, mathematical structure |
| `hep-lat` | Lattice calculations |
| `astro-ph.CO` | Cosmological implications as primary focus |

### Cross-Listing Strategy
- Always cross-list to relevant secondary categories (e.g., `hep-ph` primary + `astro-ph.CO` cross-list for dark matter with cosmological constraints)
- Cross-listing increases visibility across communities

### Timing Strategy
- **Best posting time**: Sunday evening → Monday morning (US Eastern) for maximum visibility on Monday's new listing
- **Avoid**: Friday afternoon posting (buried over the weekend)
- **Anomaly response**: Post ASAP after experimental announcement, but ensure quality — a wrong fast paper is worse than a correct slow one
- **Coordinate with experiments**: If your work responds to a specific experimental result, time the arXiv posting to coincide with or closely follow the experiment's public release
- **Version management**: v1 = submission version. Update to v2 only after significant changes (accepted version). Minor typo fixes can wait.

## 8. Figure-First Storyboard Guide

### Principle
Editors and referees look at figures first. Your 2-3 key plots must tell the entire story.

### Essential Figure Types for Particle Phenomenology
1. **Parameter Space Plot**: Allowed vs excluded regions in the model's parameter space
   - Include: current experimental bounds (shaded), projected sensitivities (dashed), your new result (solid line or band)
   - Caption must be self-contained: state what is excluded and by which experiment

2. **Distribution / Signal Plot**: Kinematic distributions or signal rates
   - Show: signal vs background, key observables, cut flow if relevant
   - Include statistical significance or expected exclusion

3. **Comparison / Summary Plot**: Your result in context of existing literature
   - Show: how your constraints compare with other approaches
   - Useful for PRL: demonstrates the "impact" visually

### Figure Checklist
- [ ] Every axis has a label with units
- [ ] Legend is readable without zooming
- [ ] Color scheme is colorblind-friendly (use line styles too)
- [ ] Each caption is self-contained (a reader should understand the figure without reading the text)
- [ ] Figures are vector format (PDF) for print quality
- [ ] Each figure has a clear "takeaway message" stated in the caption

## Brainstorming Guidance
When planning a journal submission strategy:
- Start by classifying your result using the Strategy Configuration (Section 1)
- Use the Framing Switchboard to draft journal-specific versions of your abstract
- Complete the full Referee Defense Checklist before any submission
- Pre-commit your cascade path and prepare cover letters for primary and backup journals
- Plan your arXiv submission timing around the experimental calendar
- Build your paper around the figures — if the figures don't tell a compelling story, the text won't save it

## Implementation Guidance
- Use `revtex4-2` for APS journals (PRL, PRD, PRX)
- Use `jheppub.sty` for JHEP
- Use Springer's `svjour3` class for EPJC
- Use Elsevier's `elsarticle` class for PLB and NPB
- Check word count with `texcount` — PRL's 3750 limit is strictly enforced
- Use `axodraw2` or `tikz-feynman` for Feynman diagrams
- Manage references with INSPIRE-HEP BibTeX export for consistent formatting

## Visualization Guidance
- Parameter space exclusion plots with multiple constraint overlays
- Cross-section or branching ratio plots as a function of model parameters
- Signal significance projections for current and future colliders
- Comparison plots showing improvement over existing bounds
- Use `matplotlib` with publication-quality settings or `ROOT` for HEP-standard plots
